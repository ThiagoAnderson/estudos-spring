Os conteúdos deste repositório são baseados em uma playlist prática de desenvolvimento com Spring, onde são abordados conceitos fundamentais e boas práticas utilizadas no mercado.

---

## 🚀 Objetivo

Este repositório tem como objetivo:

- Consolidar conhecimentos em **Spring Boot**
- Aprender a desenvolver **APIs REST**
- Praticar conceitos de:
  - Injeção de dependência
  - Camadas (Controller, Service, Repository)
  - Persistência com banco de dados
- Servir como base para projetos mais completos no futuro

---

## 🧠 Conteúdos Estudados

Durante os estudos, foram abordados temas como:

- ✔️ Introdução ao Spring Boot
- ✔️ Criação de projetos com Spring Initializr
- ✔️ Estrutura de um projeto Spring
- ✔️ Controllers e endpoints REST
- ✔️ Injeção de dependência (@Autowired)
- ✔️ Camada de serviço (Service)
- ✔️ Camada de repositório (Repository)
- ✔️ Integração com banco de dados (JPA)
- ✔️ CRUD completo
- ✔️ Testes básicos de API

---

## 🛠️ Tecnologias Utilizadas

- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- Banco de dados (H2 / MySQL)
- Postman / Insomnia
